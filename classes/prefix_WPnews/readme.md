**Version 1.0** (23.02.2020)

Custom class "WPnews"

## CONFIGURATION OPTIONS
* $WPnews_block_label: Label for block
* $WPnews_block_max: max entries on block list
* $WPnews_detailpage: enable detail page for cpt

## CONFIGURATION FILE
```
"WPnews": {
  "block_label": "News",
  "block_max": -1,
  "detailpage": true
}
```

## USAGE
### Wordpress Shortcode
```
[newsblock]
```
